Wordament Solver
By: Stefanus Agus Haryono/13514097

Wordament Solver ini dibuat dalam bahasa pemrograman JAVA.
Dalam penggunaan program ini, diperlukan Java Runtime Environment yang telah terinstall pada Hardware Anda.
Program ini memerlukan sebuah file teks berisi data kamus "dictionary.txt" yang akan menjadi panduan dalam
menentukan kata yang valid.
Pada program ini, pengguna dapat memasukkan tepat satu buah huruf alfabet pada setiap sel matriks wordament yang tersedia.
Karakter yang bukan berupa alfabet seperti simbol dan angka tidak diperbolehkan pada program ini.
Huruf kapital tidak memiliki perbedaan dengan huruf kecil pada eksekusi program ini.
Jika pengguna telah menentukan huruf pada setiap sel dan ingin menyelesaikan puzzle wordament tersebut, pengguna
hanya perlu menekan tombol solve.
Hasil eksekusi program dan semua solusi dari Wordament akan ditampilkan pada list di sebelah kanan matriks.
