Paduan penggunaan aplikasi Wordament Solver yang dibuat dalam rangka tugas CA-IRK,
Fairuz Astra Pratama - 13514104

1. Buat file dictionary.txt yang berisi daftar kata valid dipisah dengan baris baru di direktori program (sudah disediakan)
2. Buka program (pastikan komputer memiliki JRE versi terbaru)
3. 1. Akan ada matrix 4x4 yang merepresentasikan matrix Wordament, ubah sesuai keinginan
   2. Hanya boleh terdapat satu karakter per kotak, kapitalisasi tidak berpengaruh
4. Tekan tombol solve dibawah matrix dan tunggu
5. Program akan memulai memproses solusi dengan waktu maksimum 2 menit,
6. 1. Setelah selesai akan muncul daftar kata dan poin di kanan matrix 
   2. Scroll daftar ke bawah untuk melihat total kata dan poin yang didapat
   3. Setiap kata di daftar dapat ditekan untuk melihat jalur yang diambil dari matrix dalam bentuk daftar koordinat matrix
7. Setelah itu matrix dapat diubah dan digunakan kembali

Catatan tambahan:
1. Folder src berisi folder Netbeans program yang dapat digunakan
2. Penggunaan karakter non-romawi di dictionary dan di matrix diperbolehkan program
